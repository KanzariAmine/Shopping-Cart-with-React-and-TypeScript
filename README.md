# Shopping Cart with React and TypeScript

# Technology Used:

1. React with TS
2. https://fakestoreapi.com/ (API)

## Quick Start:

```bash
#Clone repository
https://github.com/KanzariAmine/Shopping-Cart-with-React-and-TypeScript.git

# Install dependencies
npm install

# Run App
npm start

# Build for production
npm run build
```
